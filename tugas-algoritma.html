Algoritma Sekuensial

Tugas Implementasi Algoritma Sekuensial - Menghitung Gaji Bersih Karyawan

1. Deskripsi Kasus

Sebuah perusahaan ingin membuat program sederhana untuk menghitung Gaji Bersih seorang karyawan. Diketahui bahwa Gaji Bersih didapatkan dari Gaji Pokok ditambah dengan Tunjangan (20% dari Gaji Pokok), kemudian dikurangi dengan Pajak (10% dari total Gaji Pokok dan Tunjangan).

Buatlah algoritma sekuensial yang menerima input Gaji Pokok, lalu menghitung dan menampilkan Tunjangan, Pajak, dan hasil akhir Gaji Bersih.

2. Flowchart

Representasi visual dari alur program (diagram alir) yang berjalan secara berurutan dari atas ke bawah.

graph TD A([Mulai]) --> B[/"Input Gaji Pokok"/] B --> C["Tunjangan = 0.2 * Gaji Pokok"] C --> D["Pajak = 0.1 * (Gaji Pokok + Tunjangan)"] D --> E["Gaji Bersih = Gaji Pokok + Tunjangan - Pajak"] E --> F[/"Output Gaji Bersih"/] F --> G([Selesai])

3. Pseudocode

Deskripsi langkah-langkah logika program secara terstruktur sebelum diimplementasikan ke dalam kode.

PROGRAM Hitung_Gaji_Bersih

DEKLARASI:
  gaji_pokok, tunjangan, pajak, gaji_bersih : Real

ALGORITMA:
  READ gaji_pokok
  tunjangan ← 0.2 * gaji_pokok
  pajak ← 0.1 * (gaji_pokok + tunjangan)
  gaji_bersih ← gaji_pokok + tunjangan - pajak
  PRINT "Gaji Bersih: ", gaji_bersih

4. Implementasi Python

Implementasi algoritma sekuensial di atas menggunakan bahasa pemrograman Python.

hitung_gaji.py 
# Program Menghitung Gaji Bersih Karyawan

def hitung_gaji_bersih(gaji_pokok):
    """Fungsi untuk menghitung gaji bersih secara sekuensial."""
    
    # 1. Menghitung Tunjangan (20% dari gaji pokok)
    tunjangan = 0.20 * gaji_pokok
    
    # 2. Menghitung Pajak (10% dari total gaji pokok + tunjangan)
    pajak = 0.10 * (gaji_pokok + tunjangan)
    
    # 3. Menghitung Gaji Bersih
    gaji_bersih = gaji_pokok + tunjangan - pajak
    
    return tunjangan, pajak, gaji_bersih

# Input (Simulasi input dari user)
gaji_pokok_input = 5000000 

# Proses
tunjangan, pajak, gaji_bersih = hitung_gaji_bersih(gaji_pokok_input)

# Output
print("=== Rincian Gaji Karyawan ===")
print(f"Gaji Pokok  : Rp {gaji_pokok_input:,.2f}")
print(f"Tunjangan   : Rp {tunjangan:,.2f}")
print(f"Pajak       : Rp {pajak:,.2f}")
print(f"-----------------------------")
print(f"Gaji Bersih : Rp {gaji_bersih:,.2f}")
Output Terminal
