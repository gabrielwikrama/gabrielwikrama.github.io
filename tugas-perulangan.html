Struktur Perulangan

Tugas Implementasi Perulangan (Looping) - For, While, Do-While, dan Nested Loop

1. Deskripsi Program

Program ini mendemonstrasikan empat jenis struktur perulangan yang umum digunakan dalam bahasa pemrograman, khususnya menggunakan Python:

  • For Loop: Digunakan untuk perulangan yang jumlah iterasinya sudah diketahui secara pasti dari awal.
  • While Loop: Digunakan ketika kondisi perulangan dievaluasi di awal sebelum blok kode dijalankan.
  • Do-While Loop: Python tidak memiliki struktur Do-While bawaan, sehingga disimulasikan menggunakan while True dengan kondisi break di akhir blok untuk menjamin eksekusi minimal satu kali.
  • Nested Loop (Perulangan Bersarang): Perulangan di dalam perulangan lain, sering digunakan untuk membuat pola matriks atau baris-kolom yang kompleks.

2. Implementasi Program Python

Berikut adalah kode program yang menerapkan keempat struktur perulangan tersebut secara berurutan.

perulangan.py 
print("=== 1. IMPLEMENTASI FOR LOOP ===")
# Menampilkan urutan angka dari 1 sampai 5
for i in range(1, 6):
    print(f"Perulangan For ke-{i}")

print("\n=== 2. IMPLEMENTASI WHILE LOOP ===")
# Menghitung mundur dari 3 sampai 1
hitungMundur = 3
while hitungMundur > 0:
    print(f"Menghitung mundur: {hitungMundur}")
    hitungMundur -= 1

print("\n=== 3. IMPLEMENTASI DO-WHILE LOOP ===")
# Python mensimulasikan do-while dengan while True dan break
x = 10
while True:
    print(f"Do-While dijalankan 1 kali! Nilai x = {x}")
    x += 1
    if x >= 5: # Kondisi berhenti setelah eksekusi pertama
        break

print("\n=== 4. IMPLEMENTASI NESTED LOOP ===")
# Membuat pola piramida bintang siku-siku berukuran 5 tingkat
tingkat = 5
for baris in range(1, tingkat + 1):
    polaBintang = ""
    # Loop kolom berada di dalam loop baris
    for kolom in range(1, baris + 1):
        polaBintang += "⭐ "
    print(polaBintang)
