Struktur Perulangan

Tugas Implementasi Perulangan (Looping) - For, While, Do-While, dan Nested Loop

1. Deskripsi Program

Program ini mendemonstrasikan empat jenis struktur perulangan yang umum digunakan dalam bahasa pemrograman, khususnya menggunakan JavaScript:

  • For Loop: Digunakan untuk perulangan yang jumlah iterasinya sudah diketahui secara pasti dari awal.
  • While Loop: Digunakan ketika kondisi perulangan dievaluasi di awal sebelum blok kode dijalankan.
  • Do-While Loop: Mirip dengan While, tetapi menjamin bahwa blok kode akan dieksekusi minimal satu kali, karena evaluasi kondisi dilakukan di akhir.
  • Nested Loop (Perulangan Bersarang): Perulangan di dalam perulangan lain, sering digunakan untuk membuat pola matriks atau baris-kolom yang kompleks.

2. Implementasi Program JavaScript

Berikut adalah kode program yang menerapkan keempat struktur perulangan tersebut secara berurutan.

perulangan.js 
console.log("=== 1. IMPLEMENTASI FOR LOOP ===");
// Menampilkan urutan angka dari 1 sampai 5
for (let i = 1; i <= 5; i++) {
    console.log(`Perulangan For ke-${i}`);
}

console.log("\n=== 2. IMPLEMENTASI WHILE LOOP ===");
// Menghitung mundur dari 3 sampai 1
let hitungMundur = 3;
while (hitungMundur > 0) {
    console.log(`Menghitung mundur: ${hitungMundur}`);
    hitungMundur--;
}

console.log("\n=== 3. IMPLEMENTASI DO-WHILE LOOP ===");
// Akan dieksekusi minimal satu kali meskipun kondisi di awal sudah salah
let x = 10;
do {
    console.log(`Do-While dijalankan 1 kali! Nilai x = ${x}`);
    x++;
} while (x < 5); // Kondisi bernilai false, loop langsung berhenti

console.log("\n=== 4. IMPLEMENTASI NESTED LOOP ===");
// Membuat pola piramida bintang siku-siku berukuran 5 tingkat
let tingkat = 5;
for (let baris = 1; baris <= tingkat; baris++) {
    let polaBintang = "";
    // Loop kolom berada di dalam loop baris
    for (let kolom = 1; kolom <= baris; kolom++) {
        polaBintang += "⭐ ";
    }
    console.log(polaBintang);
}
